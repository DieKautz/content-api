Search all series

This endpoint lets you search for journalistic series by name. An asterisk can serve as a wildcard or whitespace in your search query.

Endpoint

{{ api_url }}/series

Parameter

q* the main search query
fieldsuri,value,href partially select output fields
limit10 limit the amount of matches to return
offset0 offset for the list of matches

Example

Request

GET /series?q=*lorem*&limit=1 HTTP/1.1
Host: {{ api_url }}
X-Authorization: abcdefg1234xyz56789

Response

{
    "matches": [
      {
      "href": "http://www.zeit.de/serie/lorem-ipsum",
      "uri": "http://127.0.0.1:5000/api/series/lorem-ipsum",
      "value": "Lorem ipsum dolor set?"
      }
    ],
    "found": 7,
    "limit": 1,
    "offset": 0
}

Get series by ID

This endpoint provides a pre-filtered search for all articles written for a specific series. Deeper queries, pagination and partial field selection can be used to narrow down the match list.

Endpoint

{{ api_url }}/series/{id}

Parameter

q*:* the main search query
fields* partially select output fields
limit10 limit the amount of matches to return
offset0 offset for the list of matches

Example

Request

GET /series/lorem-ipsum?limit=1 HTTP/1.1
Host: {{ api_url }}
