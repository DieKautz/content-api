Willkommen zu der Beta-Version der ZEIT ONLINE Content API. Über diese Schnittstelle kann auf Inhalte und zugehörige Metadaten aus dem Archiv der ZEIT seit 1946 und von ZEIT ONLINE zugegriffen werden kann. Wir laden alle an Programmierung, Datenanalyse und Visualisierung Interessierten ein, den Index aus mehreren hunderttausenden Artikeln zu erkunden. Die Benutzung der Schnittstelle ist für nicht-kommerzielle Anwendungsfälle kostenlos. Bitte beachten Sie in jedem Falle die Nutzungsbedingungen.

Um gleich mit dem Erstellen von Anwendungen oder Auswertungen zu beginnen, können Sie sich unter Quickstart einen API-Key freischalten lassen. Der Explorer ermöglicht Ihnen die Befragung der Schnittstelle über ein einfaches Web-Interface. Mehr über die Funktionen und Endpunkte der API erfahren Sie in der englischen Dokumentation.

Wir laden Sie ein, uns Ihre Umsetzungen, Experimente und Erfahrungen mit der API via Email oder Twitter mitzuteilen. Aktuelle Informationen und spannende Projekte mit der API finden Sie auf dem ZEIT ONLINE Dev-Blog.

Welcome to the beta version of the ZEIT ONLINE content API. Via the API you can access articles and corresponding metadata from the ZEIT archive dating back to 1946 and from ZEIT ONLINE. We invite everybody interested in programming, data analysis and visualization to explore the index of hundreds of thousands articles. The API can be used for free for non-commercial use cases. Please refer to the terms of use (German only at the moment).

To get started building apps or analysing data, just request an API key in our quickstart guide. The explorer will allow you to query the API via a simple web-interface. A full documentation of the endpoints and their capabilites is also available.

We invite you to share your apps, experiments and experiences with the API with us via email and Twitter. More news and a roundup of projects using our API can be found at the ZEIT ONLINE Dev-Blog.
